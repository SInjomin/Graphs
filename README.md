# Graphs

About
----------

Graphs is opensource desktop program in which you can
create Graph and save it to the svg file.

Starting
----------
* Clone repository
  ```sh
  git clone https://github.com/SInjomin/Graphs.git
  ```

* Enter directory
  ```sh
  cd Graphs/
  ```

* Install dependencies
  ```sh
  npm install
  ```
  
* Start the program 
  ```sh
  npm start
  ```
